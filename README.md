# Surge Bandwagon Traffic

Surge iOS information panel for BandwagonHost KiwiVM traffic usage.

显示内容：

- VPS运行状态
- 本月已用流量
- 剩余流量与套餐总量
- 使用比例
- 下次流量重置时间

## 安装

在 Surge iOS 的“安装新模组”中输入：

```text
https://raw.githubusercontent.com/succichang/surge-bandwagon-traffic/main/Bandwagon-Traffic.sgmodule
```

安装后在模组参数中填写自己的 `VEID` 和 `API_KEY`。

## 安全

仓库不包含任何用户密钥。`VEID` 与 `API Key` 仅保存在用户自己的 Surge 模组参数中。

详细说明见 [Bandwagon-Traffic-使用说明.md](./Bandwagon-Traffic-%E4%BD%BF%E7%94%A8%E8%AF%B4%E6%98%8E.md)。
